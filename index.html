uncomputable
Input: a program P
Halt? ───→ YES
or NO

Assume HALT exists. Define a twisted version:

DIAGONAL(x):
  if HALT(x, x) loops forever
    return "halted"
  else
    loop forever          ← the trap

Now ask: does DIAGONAL(DIAGONAL) halt?

DIAGONAL halts → HALT says "loops" → DIAGONAL loops forever. Contradiction.
DIAGONAL loops → HALT says "halts" → DIAGONAL halts. Contradiction.

HALT cannot exist. The proof lives inside the question itself.
This is not a limitation of our tools. It is a property of computation.

incompleteness

      

        

          

            "This statement has no proof in arithmetic."
          

          

            

              arithmetical truth
            

            

              provable statements
            

            
G

          

          

            If G is provable → G is false → arithmetic is inconsistent.

            If G is true → arithmetic is consistent but incomplete.

            We cannot prove G from within arithmetic.
          

        

        

          Kurt Gödel, 1931. He encoded this as a number.

          A number that talks about its own provability.

          Arithmetic is either incomplete or inconsistent. Pick one.
